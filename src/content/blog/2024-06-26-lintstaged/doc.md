上次配置好了eslint ，在我们每次提交代码的时候就进行代码检测，这次我们写上配置lint-staged 进行代码修复。

### 安装 lint-staged

```bash
# 安装指令
pnpm add lint-staged -D
```

### 在package.json中添加lint-staged的配置


```json
"lint-staged":{
    "*.js":"eslint --fix",
    "*.ts":"eslint --fix"
}
```

修改 git hooks 钩子，每次提交的时候让他执行 **eslint --fix**


```bash
echo "npx eslint --fix" > .husky/pre-commit
```
