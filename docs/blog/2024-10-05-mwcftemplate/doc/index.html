介绍

一个基于 Monorepo + CloudFlare Worker + CloudFlare Pages + CloudFlare D1 + Prisma + NodeJs 的简易全栈项目模板。

升级日志

  • 2024-10-05 编写了基础的模板，并编写了用户注册和登录功能。

使用

# 拉取项目
git clone https://github.com/itmowang/cloudflare_system.git

# 安装pnpm
npm install pnpm -g

# 拉取项目依赖
pnpm install

# 启动本地环境
pnpm run dev

# 发布到 CloudFlare
pnpm run build
